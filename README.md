# Bot de Sinais Baccarat

Este projeto é um bot simples para análise de padrões em jogos de Baccarat (Player/Banker).

## Como usar

1. Certifique-se de ter o Python instalado.
2. Execute o script:

    ```bash
    python bot_sinais.py
    ```

3. Digite os resultados conforme eles aparecem:
    - `p` para Player (Azul)
    - `b` para Banker (Vermelho)
    - `e` para Empate
4. O bot analisará o histórico e sugerirá a próxima jogada baseada em estratégias simples (Surf, Xadrez, Dupla).
